# Emotion Based Content Recommender System 🎭📺

An AI-powered recommendation system that suggests personalized content based on the user's emotional state using multimodal emotion detection techniques.

## 🚀 Features

* Emotion detection using:

  * Facial Expressions
  * Text Sentiment Analysis
  * Voice Emotion Recognition
* Personalized content recommendations
* Machine Learning based recommendation engine
* User-friendly interface
* Real-time emotion analysis

## 🧠 Technologies Used

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Computer Vision
* OpenCV
* TensorFlow / Scikit-learn
* Flask / Streamlit

## ⚙️ How It Works

1. User provides input through camera, text, or voice.
2. The system detects the user's emotion.
3. Machine learning algorithms analyze the emotion.
4. Relevant content is recommended based on the detected mood.

## 📌 Applications

* Entertainment platforms
* Mental wellness systems
* Personalized media recommendation
* Smart AI assistants
